HTML ELEMENTS

Here you will find different HTML elements used in general in coding

Element Function
<head> Contains general information about an HTML page. The information itself is not displayed
<title> Contains the title of an HTML document. It is displayed in the browser's title bar/tab
<body> Represents the content of an HTML document. There can only be one <body> element in a document
<div> Short for "division", it divides an HTML document into sections
<h1>... <h6> Inserts headings of various sizes (h1 is the largest while h6 is the smallest)
<p> Paragraph element that contains and displays a block of text
<br> Creates a line break in text
<li> Creates a list item
<ul>, <ol> Used with <li>, <ul> creates an unordered list while <ol> creates an ordered list inside the list item
<video> Embeds a media player for video playback. Requires the src attribute for video URL
<img> Inserts an image. Requires the src attribute for image URL
<a> Short for "anchor", it creates a hyperlink in an HTML document with the href attribute
<nav> Represents a section with navigation links

HTML TABLES

Here you will find various HTML elements used in coding for creating tables

Element Function
<table> Contains content that is used to represent columns and rows of a table
<thead> Defines the headings of table columns nested in table rows
<tbody> Contains table data other than table heading and table footer content
<tfoot> Short for footer, it is used for summarizing content at the end of a table
<tr> Used for adding rows to a table
<td> Short for table data, it adds a cell of data to a table. Must be enclosed in a <tr> element
<th> Creates a heading for rows and columns of a table. Must be enclosed in a <tr> element

HTML ATTRIBUTES

Here you will find attributes and their functions

Attribute Related to element Function
src <video>, <img> Contains the URL to the video
controls <video> Displays video controls in the media player
href <a> Short for hyperlink reference, it contains a path for <a> to follow
target <a> Target value of _blank opens the hyperlink in a new tab
rowspan <th>, <td> Indicates how many rows a particular cell should span within the table
colspan <th>, <td> Indicates how many columns a particular cell should span within the table

CSS SELECTORS AND RULES

Here you will find a short introduction to CSS selectors and certain rules.

Type, Class and ID Selectors

With adding type, class or ID attributes to our code, we can use these attributes as CSS selectors. This allows us to specifically select which areas of our code we would like to stylise. We select a certain type of element by typing their name in our style.css file. By adding a period (.) before the class name in our style.css file, we select the specific element with the class attribute that we wish to modify. If we wish to use the ID attribute as a selector, we add a hashtag (#) in front of the ID name.

Selector Specificity

The selectors are ranked by their specificity. ID selectors have the highest specificity among all selectors, followed by class selectors, followed by type selectors. This means that if a class selector and an ID selector were both trying to modify the color of the text, only the ID selector would be applied (see image #1)

CSS PROPERTIES

Here you will find a short list of CSS properties and their respective functions

Property Function
font-size Used to set text sizes (normally in pixels)
font-family Used to specify a font for your text
font-weight Used to set the boldness of text (bold or normal)
text-align Used to set the text alignment of inline contents (left, right or center)
opacity Used to control the transparency of an element
background-color Used to set the background color of elements
background-image Sets the background image of an element
color Used to set color values for elements
